# Support

Use this document to decide where to get help with Gemini Security Skills.

## Installation help

Start with:

- [Install](INSTALL.md)
- [Troubleshooting](docs/troubleshooting.md)

Most installation issues are path or reload issues. Confirm the skill folders
contain `SKILL.md` files and run:

```text
/skills reload
```

## Usage help

Start with:

- [Usage](USAGE.md)
- [Skill catalog](docs/skill-catalog.md)
- [Knowledge base](KNOWLEDGE_BASE.md)

Use explicit skill names when activation is not predictable:

```text
Use soc-operations to triage this alert.
```
